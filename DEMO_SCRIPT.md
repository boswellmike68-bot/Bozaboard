# Bozaboard — Sponsor Demo Script

## Before the Demo
- Open browser to https://boswellmike68-bot.github.io/Bozaboard/ (or localhost:8080)
- Have access token ready: `steward`
- Demo takes 5-7 minutes

---

## Scene 1: The Gate (30 seconds)

**Say:** "Every interaction with the system begins at a governance gate. This isn't a login — it's a cryptographic verification. The token is hashed client-side with SHA-256. Nothing is sent to a server. Nothing leaves your browser."

**Do:** Type `steward` → click **Launch**

---

## Scene 2: The Dashboard (60 seconds)

**Say:** "This is the Boardroom. You're looking at a governance-first interface — high contrast, zero frameworks, zero dependencies, zero external calls. The 13 orbs represent consensus perspectives. The four pillars on the left are the system's structural layers."

**Do:** Point out the orb ring, the pillar buttons, the top bar (Safe Mode, Sponsor, Steward AI)

**Say:** "Safe Mode disables all animations for accessibility. This isn't a feature we added — it's foundational. Governance must never create inequality."

**Say:** "This is the BBnCC governance engine. It classifies intent, applies weighted rules, selects a Committee persona, and tracks session momentum. Every response is deterministic — same input, same output, every time."

**Type:** `proceed`
**Say:** "The Momentum Chair responds. Momentum shifts positive. Watch the counter."

**Say:** "Now the Risk Chair fires. The engine detected a risk-domain intent and selected the appropriate persona. Momentum adjusts."

**Type:** `pause`
**Say:** "Stability Chair. Momentum drops. The system doesn't blend opinions — the highest-authority rule wins."

**Type:** `proceed`
**Type:** `proceed`
**Type:** `proceed`
**Say:** "Three identical intents. The engine detects the pattern — rule weight 70 — and flags it. It's asking us to examine why we're repeating ourselves."

**Type:** `escalate`
**Say:** "Escalation rule fires at weight 95. Only the stability protocol at weight 100 outranks it. The system has a clear hierarchy of governance authority."

**Type:** `Fix the Planet`
**Say:** "Mission keyword. All 13 consensus perspectives activate. The orbs go green — the full ring is synchronized."

**Type:** `reset`
**Say:** "Session cleared. The Committee is fresh. Every session is independent and auditable."

## Scene 4: The Steward AI (60 seconds)

**Say:** "We also built a governed AI agent — the Steward. It operates under strict boundaries: it can only open pillars, show the sponsor modal, and type messages. Every action is logged in an audit trail."

**Type:** `demo`
